# Lógica de Programação - Alura

Este repositório reúne exercícios e exemplos práticos de lógica de programação desenvolvidos durante o estudo de JavaScript no curso da Alura.

## Estrutura do projeto

A pasta principal é organizada em exercícios sobre:

- Eventos e Variáveis
- Manipulação de Tabelas
- Manipulação de Valores
- TemplateString e Split

## Objetivo

Registrar e organizar os aprendizados sobre:

- uso de variáveis e eventos;
- manipulação de listas e valores;
- uso de template strings e split;
- desenvolvimento de páginas simples com JavaScript.

## Como visualizar

Abra os arquivos HTML em um navegador para ver os exemplos em funcionamento.
